Features

เรียนรู้ทักษะที่สามารถนำไปใช้งานได้จริง

เรียนรู้กระบวนการผลิตเสียงอย่างครบวงจร ตั้งแต่เริ่มอัดเสียงไปจนถึงการส่งออกไฟล์คุณภาพสูง

🎤

การอัดเสียง (Audio Recording)

เรียนรู้วิธีตั้งค่าไมโครโฟนและการบันทึกเสียงให้มีระดับสัญญาณที่เหมาะสมและเสียงใสคมชัด

✂️

การตัดต่อ (Audio Editing)

เทคนิคการตัดท่อนที่พูดผิด ลมหายใจ หรือเสียงเดดแอร์ (Dead Air) ออกได้อย่างรวดเร็วและแนบเนียน

🎵

การใส่ดนตรี (Add Music)

รู้วิธีการปรับเสียงดนตรีประกอบให้เบาลงเมื่อมีเสียงพูด (Ducking) และสร้างมิติเสียงที่น่าสนใจ

🔊

ปรับแต่งคุณภาพเสียง (Sound Quality)

เรียนรู้การใช้งาน Compressor, Equalizer (EQ) เพื่อจัดระเบียบโทนเสียงให้ฟังง่ายขึ้น

📤

ส่งออกไฟล์ (Export MP3)

เข้าใจบิตเรต (Bitrate) และการบันทึกแท็กข้อมูลเสียงเพื่ออัปโหลดขึ้นแพลตฟอร์มต่าง ๆ

🎙️

ทำพอดแคสต์ (Podcast)

รวบรวมทักษะทั้งหมดเพื่อสร้างสรรค์รายการพอดแคสต์และบทเรียนเสียงพร้อมใช้งานจริง

Curriculum

เนื้อหาหลักสูตรการเรียน

6 โมดูลที่จะเปลี่ยนคุณจากผู้เริ่มต้น สู่คนผลิตสื่อเสียงอย่างมั่นใจ

01

Module 1: Introduction

ทำความรู้จักหน้าต่างโปรแกรม Audacity เครื่องมือต่าง ๆ และโครงสร้างระบบเสียงเบื้องต้น

02

Module 2: Recording

การตั้งค่า Input/Output อุปกรณ์ไมโครโฟน และเทคนิคการอัดเสียงพูดให้ชัดเจนไร้สัญญาณรบกวน

03

Module 3: Editing

การจัดการกับ Track, การลบเสียงรบกวน, การตัดต่อท่อนที่ไม่ต้องการออก และจัดลำดับเวลาเสียง

04

Module 4: Noise Reduction

เจาะลึกฟีเจอร์เด่นในการลดเสียงฮัม เสียงจี่ และลดเสียงรบกวนฉากหลัง (Ambient Noise) อย่างเป็นธรรมชาติ

05

Module 5: Effects & Mixing

การแต่งเสียงด้วย EQ, Compressor และเทคนิคการมิกซ์เพลงประกอบกับเสียงบรรยายให้น่าติดตาม

06

Module 6: Export & Metadata

วิธีการ Export ไฟล์ในฟอร์แมตต่าง ๆ เช่น MP3, WAV และการกรอก Metadata Tag ก่อนนำไปเผยแพร่

Deliverables

สิ่งที่คุณจะได้รับในคอร์สนี้

เรียนจบพร้อมรับเครื่องมือและสิทธิพิเศษเพิ่มเติมสำหรับผู้เรียนโดยเฉพาะ

  • ✓ วิดีโอบทเรียนบันทึกย้อนหลัง (ทบทวนได้ตลอดชีวิต)
  • ✓ คู่มือประกอบการเรียน PDF Handbook สรุปย่ออ่านเข้าใจง่าย
  • ✓ ไฟล์ฝึกหัด (Practice Files) สำหรับใช้ทำตามในห้องเรียน
  • ✓ การบ้านและฟีดแบ็กแนะนำรายบุคคลจากผู้สอน
  • ✓ เกียรติบัตรรับรองการสำเร็จหลักสูตร (Certificate)
  • ✓ สิทธิ์การเข้ากลุ่มผู้เรียนพิเศษเพื่อปรึกษาและแลกเปลี่ยนความรู้

FAQ

คำถามที่พบบ่อย

เราได้รวบรวมคำถามที่ผู้สมัครมักจะสงสัยบ่อยที่สุดไว้ที่นี่

ไม่ต้องมีพื้นฐานมาก่อนเลยครับ คอร์สนี้ออกแบบขึ้นมาสำหรับผู้เริ่มต้นอย่างแท้จริง เราจะสอนการดาวน์โหลด การติดตั้งเครื่องมือ และการใช้ฟังก์ชันที่สำคัญต่างๆ แบบทีละขั้นตอนอย่างละเอียดครับ

ได้แน่นอนครับ! เรามีการบันทึกเทปวิดีโอระหว่างการสอนสดทาง Zoom และจะจัดส่งลิงก์วิดีโอให้ย้อนดูได้ตลอดชีวิตหลังจากจบการเรียนการสอน

โปรแกรม Audacity เป็นโปรแกรมใช้งานได้ฟรีอย่างสมบูรณ์แบบ รองรับทั้งระบบปฏิบัติการ Windows, macOS และ Linux ครับ ในห้องเรียนเราจะแนะนำวิธีการตั้งค่าที่ครอบคลุมทุกระบบครับ
